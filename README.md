## תכנון של פרוייקט הצגת מידע למהנדס המירוץ

#### מבנה הקבצים

הקובץ יכיל שתי קבצי מערכת (main.js | receService.js)
כאשר main יהיה הקובץ הראשי ששם יהיה הצד שבעיקר יוצג ללקוח
והקובץ השני raceService יהיה אחראי בעיקר על הבאת הנתונים מהשרת וולידציה על הנתונים

#

### תיאור קצר על המערכת

המערכת צריכה לתת תמונת מצב נוכחית למצב קיים לפי הדרישות ולהציג למשתמש את המידע הדרוש

#

### ההרצה תתבצע באמצעות הטרמינל ותרוץ בעזרת הקובץ הראשי (main.js)

#

### הנתונים שיוצגו בריצת התוכנית:

שם המערכת \
משפט פתיחה קבוע \
הדפסה על כך שהמערכת מביאה את המידע \
הדפסת הנתונים \
(שם המרוץ\
סיבוב נוכחי מתוך סיבובים סך הכל\
כמות הרכבים המשתתפים סך הכל \
כמות רכבים שסיימו כבר את המעבר ב pit stop\
פרטי הרכבים הממתינים לכניסה \
הרכב הבא בתור
)

הדפסת קריאה לנהג הבא בתור \
חיפוש מספר לפי מספר

ובסוף הודעת סיום
